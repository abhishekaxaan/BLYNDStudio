# BLYND Studio

A high-fidelity multi-disciplinary studio website built with Next.js, Tailwind CSS, Framer Motion, and Three.js.

## Features

- **Liquid Glass Aesthetic**: Blurred backgrounds, refractive effects, and fluid animations.
- **Cinematic Depth**: Custom grain overlay and mesh gradient system.
- **Interactive 3D**: React Three Fiber integrated CTA button.
- **Multi-Disciplinary Content**: Dedicated sections for 3D modeling, web/app development, and branding.
- **Mobile Optimized**: Responsive design across all architectural modules.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion
- **3D Engine**: React Three Fiber / Three.js
- **Icons**: Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the development build.
